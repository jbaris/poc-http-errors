# POC of http-errors

A simple Proof of concepts to see how [http-errors](https://github.com/jshttp/http-errors#readme) works.

## How to run

    $ git clone https://github.com/jbaris/poc-http-errors.git
    $ cd http-errors
    $ npm install
    $ node index.js

## How to test
Open in your web browser:
http://localhost:3000/
